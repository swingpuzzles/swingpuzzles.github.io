# Legal Pages for Swing Puzzles

This directory contains the essential legal documents for the Swing Puzzles application.

- User rights and data control options

### 2. Terms of Service (`terms-of-service.html`)

### 3. Cookie Policy (`cookie-policy.html`)
- Detailed explanation of cookie usage

- Responsive design that hides during fullscreen
- Consistent styling across pages
